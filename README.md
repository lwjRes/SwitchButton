##添加自定义属性
###SwitchButton

| 属性           | 含义             | 类型          |默认值         |
|:------------- |:---------------:|:-------------:|--------------|
| mask_color    | 滑块颜色 |         color|     0xFFFFFF         | 
| open_color    | 选中的背景色        |    color|        0xFF33A63A      |
| close_color   | 未选中的背景色       |       color |      0x667f7f7f        |
| checked       | 是否选中        |           boolean |       false       |
